# Study Timetable Planner

Live demo: https://timetable-nu-eosin.vercel.app

This project is a responsive, browser-based study timetable for a 3-month learning plan. It organizes study blocks from 9:00 a.m. to 10:00 p.m., tracks progress, supports catch-up planning, and includes a lightweight assistant panel for study guidance.

## Features

- 3-month study timetable with structured daily blocks
- Responsive layout for phone, tablet, and desktop
- Progress tracking and daily focus summary
- Catch-up mode for missed work and backlog items
- Floating date jump control to open any day directly
- Custom calendar extensions for adding extra topics beyond the original plan
- DSA deep-dive links for roadmap-based study support
- Assistant panel with local guidance and optional Gemini key support

## Files

- `3month_study_timetable_june_sep_2026.html` - main application file
- `vercel.json` - static deployment rewrite for Vercel

## Run Locally

Open `3month_study_timetable_june_sep_2026.html` directly in your browser.

If you prefer a local server, run a simple static server in this folder, then open the shown URL.

## Deployment

This project is deployed on Vercel as a static site.

Live URL: https://timetable-nu-eosin.vercel.app

## Notes

- Study progress and calendar extensions are stored locally in the browser.
- The planner is designed to continue beyond the initial 3-month schedule by adding custom follow-up topics into the calendar.
